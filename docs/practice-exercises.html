8  Practice Exercises

Keywords

neural networks, backpropagation, regression, classification, sigmoid, gradient descent, machine learning, educational book

8.1 Why this chapter exists

This chapter gives you a place to test whether you can do the core calculations yourself.

Reading a worked example can create the illusion of understanding.

Real understanding appears when you can:

  • identify the correct formula,
  • substitute the correct values,
  • compute the intermediate steps,
  • explain why each step is valid.

Use paper, pencil, and a calculator.

Try each exercise by hand before checking with Python.

8.2 How to use these exercises

For each exercise:

  1. write down the known values,
  2. write down the formula before substituting numbers,
  3. show each intermediate step,
  4. box your final answer,
  5. explain in one sentence what the answer means.

Do not skip directly to code.

8.3 Part A: concept and symbol checks

8.3.1 Exercise A1: identify the symbols

In your own words, define each symbol:

  1. \(x\)
  2. \(w\)
  3. \(b\)
  4. \(z\)
  5. \(\hat{y}\)
  6. \(y\)
  7. \(L\)
  8. \(\eta\)

8.3.2 Exercise A2: explain the difference

Explain the difference between:

  1. \(z\) and \(\hat{y}\)
  2. error and loss
  3. regression output and classification output
  4. weight and bias

8.3.3 Exercise A3: activation choice

State which activation is more appropriate in each case and explain why:

  1. predicting house price
  2. predicting whether an email is spam
  3. predicting annual sales
  4. predicting whether a customer clicks an ad

8.4 Part B: regression forward-pass exercises

In these exercises, use identity activation:

\[ \hat{y} = z = w_1 x_1 + w_2 x_2 + b \]

and squared error loss:

\[ L = \frac{1}{2}(y - \hat{y})^2 \]

8.4.1 Exercise B1

Given:

\[ x_1 = 1,\quad x_2 = 3,\quad y = 5 \]

and:

\[ w_1 = 1,\quad w_2 = 0.5,\quad b = 0 \]

Compute by hand:

  1. the prediction \(\hat{y}\),
  2. the error \(y - \hat{y}\),
  3. the loss \(L\).

8.4.2 Exercise B2

Given:

\[ x_1 = 2,\quad x_2 = 1,\quad y = 4 \]

and:

\[ w_1 = 0.5,\quad w_2 = 1.5,\quad b = -0.5 \]

Compute by hand:

  1. the prediction \(\hat{y}\),
  2. the error \(y - \hat{y}\),
  3. the loss \(L\).

8.4.3 Exercise B3

Given this tiny dataset:

Example \(x_1\) \(x_2\) \(y\)
1 1 0 2
2 0 2 3
3 2 1 5

and parameters:

\[ w_1 = 1,\quad w_2 = 1,\quad b = 0 \]

Compute by hand for all three examples:

  1. the prediction,
  2. the loss for each example,
  3. the mean loss across the dataset.

8.5 Part C: classification forward-pass exercises

In these exercises, use sigmoid activation:

\[ \hat{y} = \sigma(z) = \frac{1}{1 + e^{-z}} \]

and binary cross-entropy loss:

\[ L = -\left[y \log(\hat{y}) + (1-y)\log(1-\hat{y})\right] \]

Use threshold 0.5 for the predicted class.

8.5.1 Exercise C1

Given:

\[ x_1 = 1,\quad x_2 = 0,\quad y = 1 \]

and:

\[ w_1 = 1,\quad w_2 = -1,\quad b = 0 \]

Compute by hand:

  1. the linear score \(z\),
  2. the prediction \(\hat{y}\),
  3. the predicted class,
  4. the loss.

8.5.2 Exercise C2

Given:

\[ x_1 = 0,\quad x_2 = 1,\quad y = 0 \]

and:

\[ w_1 = 2,\quad w_2 = -1,\quad b = -0.5 \]

Compute by hand:

  1. the linear score \(z\),
  2. the prediction \(\hat{y}\),
  3. the predicted class,
  4. the loss.

8.5.3 Exercise C3

Given the dataset:

Example \(x_1\) \(x_2\) \(y\)
1 0 0 0
2 1 0 1
3 0 1 0
4 1 1 1

and parameters:

\[ w_1 = 2,\quad w_2 = -1,\quad b = -0.5 \]

Compute by hand for all four examples:

  1. the score \(z\),
  2. the prediction \(\hat{y}\),
  3. the predicted class.

Then state how many of the four examples are classified correctly.

8.6 Part D: regression backpropagation exercises

Use:

\[ \hat{y} = w_1 x_1 + w_2 x_2 + b \]

\[ L = \frac{1}{2}(y - \hat{y})^2 \]

8.6.1 Exercise D1

Given:

\[ x_1 = 2,\quad x_2 = 1,\quad y = 5 \]

and initial parameters:

\[ w_1 = 0.5,\quad w_2 = 1.0,\quad b = 0 \]

with learning rate:

\[ \eta = 0.1 \]

Compute by hand:

  1. the prediction \(\hat{y}\),
  2. the loss \(L\),
  3. \(\frac{\partial L}{\partial \hat{y}}\),
  4. \(\frac{\partial L}{\partial w_1}\),
  5. \(\frac{\partial L}{\partial w_2}\),
  6. \(\frac{\partial L}{\partial b}\),
  7. the updated parameters,
  8. the new prediction,
  9. the new loss.

8.6.2 Exercise D2

Repeat Exercise D1, but change the learning rate to:

\[ \eta = 0.5 \]

Then answer:

  1. did the loss still decrease?
  2. did it decrease more or less than before?
  3. what does this suggest about learning-rate choice?

8.7 Part E: classification backpropagation exercises

Use:

\[ z = w_1 x_1 + w_2 x_2 + b \]

\[ \hat{y} = \sigma(z) \]

\[ L = -\left[y \log(\hat{y}) + (1-y)\log(1-\hat{y})\right] \]

and the simplification:

\[ \frac{\partial L}{\partial z} = \hat{y} - y \]

8.7.1 Exercise E1

Given:

\[ x_1 = 1,\quad x_2 = 1,\quad y = 1 \]

and initial parameters:

\[ w_1 = 2,\quad w_2 = -1,\quad b = -0.5 \]

with learning rate:

\[ \eta = 0.1 \]

Compute by hand:

  1. the score \(z\),
  2. the prediction \(\hat{y}\),
  3. the loss,
  4. \(\hat{y} - y\),
  5. the gradients for \(w_1\), \(w_2\), and \(b\),
  6. the updated parameters,
  7. the new prediction,
  8. the new loss.

8.7.2 Exercise E2

Given:

\[ x_1 = 0,\quad x_2 = 1,\quad y = 0 \]

and initial parameters:

\[ w_1 = 2,\quad w_2 = -1,\quad b = -0.5 \]

with learning rate:

\[ \eta = 0.1 \]

Compute by hand:

  1. the score \(z\),
  2. the prediction \(\hat{y}\),
  3. the loss,
  4. \(\hat{y} - y\),
  5. the gradients for \(w_1\), \(w_2\), and \(b\),
  6. the updated parameters.

Then explain why a parameter gradient is zero only when its corresponding input contribution makes the derivative zero, such as when that input is zero in this one-neuron setup.

8.8 Part F: short answer checks

These are not full worked solutions. They are quick checks so you can detect major arithmetic mistakes.

8.8.1 Check for Exercise B1

You should get:

\[ \hat{y} = 2.5 \]

8.8.2 Check for Exercise B2

Your prediction should be:

\[ \hat{y} = 2.0 \]

8.8.3 Check for Exercise C1

You should get:

\[ z = 1 \]

and:

\[ \hat{y} \approx 0.7311 \]

8.8.4 Check for Exercise C2

You should get:

\[ z = -1.5 \]

and predicted class:

\[ 0 \]

8.8.5 Check for Exercise D1

Your initial prediction should be:

\[ \hat{y} = 2 \]

8.8.6 Check for Exercise E1

Your initial score should be:

\[ z = 0.5 \]

8.9 How to diagnose mistakes

If your answer is wrong, check these in order:

  1. Did you write the correct formula before substituting?
  2. Did you copy the numbers correctly?
  3. Did you apply the activation function correctly?
  4. Did you square the error when required?
  5. Did you use the correct loss for the task?
  6. Did you keep the gradient sign correct?
  7. Did you subtract the gradient in the update step?

8.10 Chapter checklist

By the end of this chapter, you should be able to do all of the following without looking back:

  1. compute a regression forward pass,
  2. compute a classification forward pass,
  3. compute squared error loss,
  4. compute binary cross-entropy loss,
  5. compute regression gradients by hand,
  6. compute classification gradients by hand using \(\hat{y} - y\),
  7. explain why a loss decreased after one update.

8.11 Note to lecturers

These exercises are intentionally small and board-friendly.

They are suitable for:

  • classroom hand calculation,
  • tutorial discussion,
  • short quizzes,
  • concept reinforcement before coding labs.

A strong teaching method is:

  1. let students solve individually,
  2. compare answers in pairs,
  3. explain each step aloud as a class,
  4. only then verify with Python.
