12  References

Keywords

neural networks, backpropagation, regression, classification, sigmoid, gradient descent, machine learning, educational book

12.1 Why this chapter exists

This chapter lists the main sources that support the concepts used in this book.

The goal of this mini-project is first-principles understanding, not literature review.
Still, a teaching book should point learners and lecturers to standard references for deeper study.

12.2 Core references used in spirit

The ideas in this book are standard neural-network foundations, especially:

  • weighted linear combination,
  • activation functions,
  • squared error loss,
  • binary cross-entropy loss,
  • gradient descent,
  • backpropagation via the chain rule.

12.3 Reference list

Bishop, C. M. (2006). Pattern recognition and machine learning. Springer.
Goodfellow, I., Bengio, Y., & Courville, A. (2016). Deep learning. MIT Press. https://www.deeplearningbook.org/
Haykin, S. (2009). Neural networks and learning machines (3rd ed.). Pearson.
Rumelhart, D. E., Hinton, G. E., & Williams, R. J. (1986). Learning representations by back-propagating errors. Nature, 323(6088), 533–536. https://doi.org/10.1038/323533a0

12.4 How to use these references

Use these references in different ways depending on your goal:

  • for intuition and broad overview, start with a textbook chapter,
  • for mathematical detail, study the derivations carefully,
  • for implementation perspective, compare the formulas here with software-library documentation,
  • for teaching, use the references to connect this hand-calculable book to the broader neural-network literature.

12.5 Note to learners

You do not need to master all these sources before using this book.

The main job of this page is to give you trustworthy places to go next after you understand the worked examples in this book.
